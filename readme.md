# Air Q detect

โปรแกรมวิเคราะห์คุณภาพอากาศ อ่านข้อมูลค่า PM2.5 รายวัน/รายเดือน จากไฟล์แล้วนำมาวิเคราะห์ระดับความรุนแรง แนวโน้ม และความเสี่ยงต่อสุขภาพ มีเกณฆ์การตัดสินที่จามค่าฝุ่นจริง มีการแสดงค่าเฉลี่ย ค่าสูงสุด/ต่ำสุด วันที่ค่าสูงสุดในรอบ สัปดาห์/เดือน หรือสูงสุดในรอบวัน จำนวนวันที่ค่าฝุ่นเกินมาตรฐานในรอบ สัปดาห์/เดือน ค่าเฉลี่ยรายเดือน เปรียบเทียบเดือนที่แล้วกับเดือนปัจจุบัน หาเดือนที่แย่ที่สุดในรอบปี วิเคราห์แนวโน้มว่าผุ่นจะมากขึ้นหรือน้อยลง วิเคราห์วันที่มีหมอกควัน แจ้งเตือนเกี่ยวกับสุขภาพในวันที่อากาศแย่ และเตือนภัยให้ใส่หน้ากาก

ในด้านโปรแกรมจะมีการอ่านค่าจากทั้งไฟล์ .txt, .xlsx และ/หรือ อ่านค่าจาก sensor และ/หรือ สามารถรับค่าจาก online maps (Airnow, Airly ถ้าเป็นไปได้) ซึ่งข้อมูลภายในไฟล์จะประกอบด้วยวันที่ เวลา และค่าความเข้มข้นของฝุ่น PM2.5 หลังจากนำเข้าข้อมูลแล้ว โปรแกรมจะดำเนินการตรวจสอบความถูกต้องของข้อมูลเบื้องต้น เช่น การตรวจหาค่าที่ขาดหาย ข้อมูลที่ซ้ำซ้อน หรือค่าที่อยู่นอกช่วงที่สมเหตุสมผล เพื่อเพิ่มความน่าเชื่อถือของผลการวิเคราะห์

เมื่อได้ข้อมูลที่ผ่านการตรวจสอบแล้ว โปรแกรมจะทำการวิเคราะห์ระดับคุณภาพอากาศ โดยนำค่าฝุ่น PM2.5 มาเปรียบเทียบกับเกณฑ์มาตรฐานคุณภาพอากาศที่กำหนดไว้ตามหลักวิชาการหรือมาตรฐานสิ่งแวดล้อม ผลลัพธ์ที่ได้จะแสดงระดับคุณภาพอากาศในแต่ละช่วงเวลา เช่น ระดับอากาศดี ระดับปานกลาง ระดับเริ่มมีผลกระทบต่อสุขภาพ และระดับที่มีผลกระทบต่อสุขภาพอย่างรุนแรง และยังมีบอกเป็นตัวเลยแบบ micrograms per cubic meter (µg/m³)

นอกจากนี้โปรแกรมยังสามารถคำนวณสถิติพื้นฐานของข้อมูล PM2.5 เพื่อใช้ในการวิเคราะห์เชิงปริมาณ เช่น ค่าเฉลี่ย ค่าสูงสุด ค่าต่ำสุด และค่ามัธยฐาน เพื่อสรุปคุณภาพของอากาศในแต่ละวัน สัปดาห์/เดือน/ปี และเพื่อความงาสยในการเปรียบเที่ยบข้อมูลใหม่กับข้อมูลเก่า โปรแกรมจะมีการตรวจจับเหตุการณ์ผิดปกติของฝุ่น เช่น ค่าฝุ่นที่สูงเกินกว่าค่าเฉลี่ยมากๆ หรือการเปลี่ยนแปลงของค่าฝุ่นอย่างรวจเร็วในระยะเวลาสั้นๆ
